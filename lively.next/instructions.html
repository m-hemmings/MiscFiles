Instructions for NBA Visualization

The intent of this project is to show how it is possible to easily create a meaningful visualization using the vis.js library.

This visualization contains the following:

  • One node for each player (contains their statistics in various properties)
  • One node for each team, divided by Western and Eastern Conference
  • One node for each conference, attached to the league node
  • One node for the league

The following listeners exist for this vis.js morph:
Listener Name Function
Single Click Currently not assigned to a function, previously used to center graph on a clicked node
Double Click
  • If used on a node that is a cluster container of other nodes, it will expand/collapse
  • If used on a player node, it will open an editor in which the various values can be edited by the user
Long Click
  • If used on a node, it will create a copy with no attached edges and append a "_1" to both the name and node id
  • If used on empty space, this will create a node that has no values in it with a unique name and node id
